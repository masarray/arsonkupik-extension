Policy · Effective July 24, 2026

Privacy is part of the signal path.

ArSonKuPik 0.3.112 processes selected browser-tab audio locally and is intentionally designed without analytics, advertising, accounts, or cloud audio transport.

Summary

ArSonKuPik processes selected browser-tab audio locally. It does not upload, record, retain, sell, or share audio. Before enhancement can start, the extension presents a local-processing notice and requires an explicit user action.

First-use notice

The popup explains that tab audio remains on the device and that a normalized site hostname and per-site enhancement preference may be stored locally when that feature is used. The accepted notice version and acceptance time are stored locally so a materially changed notice can require renewed consent.

Audio processing

Audio capture starts only after the user chooses a normal HTTP or HTTPS tab and starts enhancement. Chrome supplies that tab's audio stream to an offscreen extension document, where the Web Audio API applies equalization, dynamics, harmonic color, stereo shaping, gain, limiting, metering, and system-default playback.

Audio is not sent to a server and is not used for advertising, profiling, analytics, model training, or any unrelated purpose.

Information handled locally

ArSonKuPik may read the active or captured tab's URL and title to identify the user-selected source. It derives only the normalized hostname, such as example.com, for a saved per-site enhancement preference.

  • Audio-engine settings and selected preset.
  • Custom preset names and values.
  • A normalized hostname associated with an optional per-site enhancement preference.
  • Preference update times and privacy-consent metadata.
  • A manual English or Bahasa Indonesia language preference.
  • Active capture state needed to coordinate the service worker and offscreen document.

The extension does not store page paths, query strings, saved per-site tab titles, saved per-site tab identifiers, page content, forms, passwords, messages, cookies, or account credentials.

Permission use

PermissionPurpose
activeTabLimits user-triggered access to the tab the user currently selects.
tabCaptureObtains audio from the tab the user explicitly chooses to enhance.
offscreenHosts the continuous local Web Audio graph after the popup closes.
storageSaves settings, presets, consent, language choice, and per-site enhancement preferences locally.

ArSonKuPik declares no host permissions and does not request microphone or location access. Playback uses the browser and operating system default audio output; no speaker identifier or named output route is stored.

Network and third parties

The extension runtime has no remote application code, analytics SDK, advertising SDK, telemetry endpoint, cloud synchronization, account system, or ArSonKuPik web-service request. Links opened by the user are normal browser navigation and are subject to the destination's own policy.

Optional development support

ArSonKuPik can open an optional public support page only after the user selects Support ArSonKuPik. The extension does not open it automatically and does not send audio, browsing activity, settings, identifiers, or payment information. QRIS payment is completed outside the extension in the user's chosen banking or e-wallet application. All core features remain available whether or not the user contributes.

Retention and deletion

Local configuration remains in the current Chrome profile until the user changes or deletes it. The popup provides two direct controls:

  • Clear Site Preferences removes per-site enhancement preferences while keeping custom presets and general settings.
  • Reset All Local Data stops capture and removes settings, custom presets, site preferences, language preference, and privacy consent.

Uninstalling the extension also removes Chrome-managed extension data according to browser behavior.

Chrome Web Store Limited Use

Use of information received from Chrome APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements. Data is used only to provide and improve the user-facing audio-enhancement functionality described in the extension listing.

Children's privacy

ArSonKuPik is a general-purpose audio utility and does not knowingly collect information from children or any other user.

Policy changes

Material changes will be documented in the repository and reflected by an updated effective date or privacy-notice version. A material change can require renewed consent before enhancement starts.

Contact

For privacy or security concerns, use the repository's private vulnerability-reporting feature. Do not include private account data or sensitive personal information in a public issue.
