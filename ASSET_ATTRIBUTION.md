# Asset Attribution

ArSonKuPik does not bundle third-party product photography or a sponsored product poster in the extension runtime.

The bundled interface icons, screenshots, and project artwork are maintained as ArSonKuPik project assets. A maintainer must verify ownership or an appropriate redistribution license before adding any externally sourced image, font, audio sample, logo, product photograph, or other creative asset.

Every future third-party asset must record:

- Source and author.
- License and version.
- Required attribution text.
- Whether modification and redistribution are permitted.
- The exact files in which it is used.

Assets without verifiable redistribution rights must not be included in Chrome Web Store packages or GitHub releases.


## Merchant QRIS image

`docs/assets/qris-arsonkupik.png`, when enabled, must be supplied by the project owner from an authorized merchant QRIS provider. It is intentionally not included in the default repository package. See [QRIS_SETUP.md](QRIS_SETUP.md).
